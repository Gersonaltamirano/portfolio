# Senior Web Developer Portfolio

Static portfolio created for Gerson Altamirano's senior web developer applications.

## Edit before sending

- Add an email address if you want one displayed in the contact section.
- If you have screenshots or official logos you are allowed to use, place them in `assets/` and swap the project media blocks.
- Add live URLs or private demo notes to each project if available.

## Run locally

Open `index.html` directly in the browser, or visit `http://localhost/portafolio/` if Laragon is serving `c:\laragon\www`.
